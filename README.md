# [article-app]
article 검색 및 관리 API

## GET /articles
: article 리스트 조회

## GET /articles/{articleNumber}
: article 1개 조회

## POST /articles
: article 등록

## PUT /{articleNumber}
: article 수정

## DELETE /{articleNumber}
: article 삭제
